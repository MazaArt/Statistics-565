\section*{Introduction}
This final report completes my replication and extension project on \textit{Topological Data Analysis of Financial Time Series: Landscapes of Crashes} by Marian Gidea and Yuri Katz (\url{https://arxiv.org/pdf/1703.04385}). As in the proposal and intermediate write-up, the central question is whether topological summaries of rolling multivariate return clouds provide useful early-warning behavior before major market stress events.

The project evolved in two stages. First, I replicated the paper's core US-equity setup and event-focused diagnostics around dot-com and Lehman windows. Second, I extended the same pipeline beyond the paper sample, with post-2016 data and a dedicated COVID-era stress test. The report is organized accordingly: replication outcomes first, then extension outcomes, then a short conclusion on what the exercise supports and where limits remain.

\section*{Replication: Setup and Main Findings}
\subsection*{Data and pipeline}
Replication uses Yahoo Finance adjusted close data for:
\begin{multicols}{2}
\begin{itemize}
    \item S\&P 500 (\texttt{\^{}GSPC})
    \item DJIA (\texttt{\^{}DJI})
    \item NASDAQ Composite (\texttt{\^{}IXIC})
    \item Russell 2000 (\texttt{\^{}RUT})
\end{itemize}
\end{multicols}
with VIX (\texttt{\^{}VIX}) as an external benchmark. The implementation follows the original paper's logic:
\begin{enumerate}
    \item Compute daily log-returns \(r_{i,j} = \log(P_{i,j}/P_{i-1,j})\).
    \item Build rolling point clouds in \(\mathbb{R}^4\) using windows \(w\in\{50,100\}\).
    \item Compute \(H_1\) persistent homology via Vietoris--Rips filtration.
    \item Convert persistence diagrams into persistence landscapes and track \(L^1\), \(L^2\) norms.
    \item Run pre-event trend diagnostics (variance, low-frequency spectral power, Kendall-\(\tau\)-style trend tests).
\end{enumerate}

The quantitative trend checks are also informative. In this run, the Lehman pre-window strongly supports increasing-risk behavior (positive and highly significant trend metrics), while dot-com is significant but directionally mixed for some diagnostics. This does not invalidate the topological approach; it highlights event heterogeneity and the importance of interpreting TDA signals as part of an indicator set rather than a universal one-direction trigger.

\section*{Extension: Post-Paper Sample and COVID-Era Test}
\subsection*{Extension design}
To move beyond the original paper horizon, I reused the same processing pipeline on an extended sample. This preserves methodological comparability while testing whether the same topological indicators remain informative in newer market regimes.

The extension data cover the post-paper period into 2020 and include:
\begin{itemize}
    \item persistence landscape norm series (\(w=50\), \(w=100\)),
    \item aligned 4-index returns,
    \item VIX benchmark.
\end{itemize}

\subsection*{Post-paper behavior (2017 onward)}
Figure~\ref{fig:ext_postpaper} compares standardized \(L^1\) landscape norm and standardized VIX through the extension sample. Broadly, the two series co-move in stretches of higher stress, but alignment is not universal. The COVID-era close-up (Figure~\ref{fig:ext_covid}) is the main counterexample here: \(L^1\) stays low going into the crash even as risk indicators move, so the extended sample supports reading the norm as regime-sensitive rather than uniformly leading.

\subsection*{COVID-era stress test}
The COVID shock is an out-of-sample stress environment relative to the original paper. In the figure for this window, the normalized \(L^1\) landscape norm is not elevated before the crash: it sits near the bottom of its range in the weeks leading into the February--March 2020 selloff (in some stretches, close to the lowest values in the plotted window). So this episode does not reproduce the Lehman-style pattern of rising topological complexity \emph{ahead} of the breakpoint. Volatility benchmarks such as VIX tell a different story in the same graphic, highlighting a clear divergence between implied fear and this particular \(L^1\) summary of return-cloud topology.

That pattern is plausible if the epidemic was largely an unforeseen, fast repricing shock rather than a slow multivariate deformation of daily return geometry that homology would accumulate before prices broke. Interpreted honestly, COVID counts as evidence that topological landscape norms need not anticipate every crisis mechanism, and that exogenous-hit episodes can violate the gradual-buildup intuition the dot-com/Lehman windows partly support. Useful follow-ups would shorten the window \(w\), add higher-frequency covariates, or stress-test diagnostics specifically tailored to abrupt breaks; the takeaway here is methodological caution, not reaffirmation of a uniform early-warning narrative.

\section*{Conclusion}
Overall, the project supports parts of Gidea and Katz's qualitative story: persistence landscape norms can track meaningful deformations of multivariate return geometry around stress, with clearest reinforcement in replication around the Lehman episode. The post-paper extension does not uniformly extend that narrative: around COVID, the \(L^1\) norm shown in Figure~\ref{fig:ext_covid} is subdued immediately before the crash, which cautions against treating any single topological time series as a reliable pre-crash alarm across mechanisms (gradual imbalances versus sudden exogenous shocks).

At the same time, this project also reinforces practical limits. Signal direction and strength are event-dependent, parameter choices matter, and topological indicators are best treated as complements to classical diagnostics (especially volatility-based benchmarks like VIX), not as standalone forecasting devices. The most defensible applied use is as one component in an ensemble risk-monitoring framework with explicit out-of-sample validation.
